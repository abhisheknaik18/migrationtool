Quick Reference - NPM Scripts
==============================

Frontend (Run from root directory):
  npm run dev          - Start development server (port 5173)
  npm run build        - Build for production
  npm run preview      - Preview production build
  npm run lint         - Run ESLint

Backend (Run from root directory):
  npm run backend      - Start backend server in dev mode (port 3001)
  npm run backend:install - Install backend dependencies

Backend (Run from backend/ directory):
  cd backend
  npm run dev          - Start server with auto-reload
  npm run build        - Compile TypeScript to JavaScript
  npm start            - Run compiled production server
  npm run db:push      - Initialize/migrate database

Installation Steps:
===================
1. npm install
2. npm run backend:install
3. cd backend && npm run db:push && cd ..
4. npm run backend (in one terminal)
5. npm run dev (in another terminal)
6. Open http://localhost:5173

Default Ports:
==============
Frontend: http://localhost:5173
Backend:  http://localhost:3001
API:      http://localhost:3001/api
